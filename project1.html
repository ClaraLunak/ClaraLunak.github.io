Twitter Fake News

Om projektet

Maskininlärningsmodell för att identifiera falska nyheter på Twitter, med fokus på datakvalitet och precision.

Tekniker

- Python

- scikit-learn

- matplotlib

- Pandas

- Twitter API

Utmaningar

Rensa och bearbeta stora mängder textdata, balansera dataset, undvika överanpassning av modellen.

Resultat

Modellen klassificerar nyheter med hög noggrannhet och kan enkelt integreras i större applikationer.

Min roll

Jag ansvarade för datainsamling, preprocessing, modellträning och utvärdering.

Framtida förbättringar

Integrera realtidsdata och utveckla ett webbaserat gränssnitt för användarvänlighet.
